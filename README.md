# Recipe App

## Overview

The Recipe App is a modern React-based web application that helps users discover and explore a vast collection of recipes. Built with React 19, Vite, and styled with Tailwind CSS, it leverages the TheMealDB API to provide real-time access to thousands of meal recipes from various cuisines and categories. Users can search for specific recipes, browse by meal categories, and view detailed recipe information including ingredients and cooking instructions. The app features an intuitive search interface and category filtering system that makes recipe discovery seamless and enjoyable.

The application includes a comprehensive favorites management system that allows users to save their preferred recipes locally using browser storage, enabling quick access to bookmarked meals. With smooth animations powered by the Motion library and an elegant UI built with Lucide React icons, the app provides a polished user experience. The frontend uses React Router for seamless page navigation between the home recipe browsing interface and a dedicated favorites page, making it easy for users to organize and manage their collection of favorite recipes.
